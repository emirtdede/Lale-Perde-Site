# Proje Hafızası

Son güncelleme: 2026-06-21

## Proje Özeti
LALE PERDE adındaki premium perde mağazası için dinamik bir vitrin, ölçü hesaplama ve admin yönetim paneli içeren, e-ticaret satışı olmayan ancak WhatsApp ile teklif alma odaklı modern bir web sitesi projesidir.

## Teknik Özet
- Frontend & Backend: Next.js (App Router, React 19, TypeScript)
- Stil: Vanilla CSS (CSS Modules & Global CSS)
- Dil: Çoklu dil desteği (Türkçe & İngilizce), i18n motoru
- Form Yönetimi: Formspree (İletişim formları için)
- Veri Yönetimi: Client-side dynamic state (localStorage ile mock DB)

## Şu Anki Durum
- Tamamlanan: Next.js projesi sıfırdan kuruldu.
- Devam eden: Proje planlama ve dokümantasyon aşaması.
- Sıradaki: PRD, SRS, SAD belgelerinin hazırlanması ve onay sonrası kodlamaya başlama.

## Son Yapılan İş
- create-next-app ile proje bootstraplendi, env/rules dosyaları taşındı.

## Dikkat Edilmesi Gerekenler
- CSS yapısında kurumsal lacivert ve altın renk paleti kullanılacaktır.
- Hardcoded metin kullanılmayacak, dil dosyalarına eklenecektir.
- Admin panelinde kimlik bilgileri doğrulaması SMS/Email simülatörüyle yapılacaktır.
